Delivered in the userInfo of the status change notification.

#define BFG_LOGIN_STATUS_CHANGE_KEY_PREVIOUS_HMID

Delivered in the userInfo of the status change notification.

Returns the HMID of the previous user if the app is re-entered with a different user active than when the app was backgrounded. Otherwise this value is undefined.
