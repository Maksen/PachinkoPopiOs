Introduction to the Big Fish iOS SDK.

The Big Fish iOS Software Development Kit

This document describes the specific practices for developing games for Big Fish.

The Big Fish iOS Software Development Kit (Big Fish iOS SDK) provides resources for developers to use when they create iOS apps for the Big Fish Game Network.

These resources supplement the resources that Apple provides in the iOS Developer Library.

Developers can find the Big Fish iOS SDK and related resources in an external (SVN) repository, as well as current documentation online at Big Fish Developer Documentation.

You should be familiar with general iOS development practices before attempting to use the Big Fish iOS SDK.

SDK Resource Directories

The SDK contains the following top-level directories:

  • bfg_gamespecific_resources
  • docs
  • examples
  • framework
