Detailed Description

Required game logging APIs.

This class is used for reporting all game events.

logCustomEvent logs a custom event.

Parameters
nameThe name of the event at its lowest level of detail.
valueThe value of this parameter is dependent on eventDetails1.
levelThis parameter can be used to track game levels where custom events occur. If it is not applicable to your event, you should use 1.
details1This field should be outlined in a document provided by your producer.
details2Typically reflects the "type" of an event occurring within details1.
details3Typically describes the method or location of an event.
additionalDetailsFor reporting purposes, any additional data pertaining to the event can be added as a dictionary. This must be a flat dictionary containing only NSString * as keys and values.
