iOS Facebook Post Layout

There are two kinds of posts you will want to use with Facebook, basic posts and brag posts. They both employ the Facebook news feed feature. For more information, see the Facebook iOS Games Tutorial, Bragging and News Feeds.

Note
The icon image must be hosted by the developer. Any size can be used initially, but it will be automatically resized to 90px x 90px when posted to the Facebook wall.
