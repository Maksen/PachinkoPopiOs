What Is Freemium?

Freemium is a version of a game that follows the "Try Before You Buy" model. Unlike a typical “lite“ or “trial“ version, all content for the full game is included in a Freemium app. This full content can be unlocked by the user via an In-App Purchase. The user is prompted to make this purchase from a button on the Main Menu, and through an upsell message that appears after the free content has been completed.

The Freemium model reduces the number of steps required for a user to purchase full game content after trying a game. There is a significant increase in conversions from "trial" to "full" when an app is offered as a Freemium model. Additionally, every free download offers exposure to the game, helping to increase the number of potential customers.

Big Fish Games releases Freemium and Free-to-Play iOS apps created by third-party developers, in addition to its own Big Fish Studios games.
